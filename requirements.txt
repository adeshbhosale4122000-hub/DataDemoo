🚀 Data Pipeline Dashboard
📌 Overview
This project is a full-stack data pipeline application that allows users to upload CSV files, process them through a data pipeline, and download the transformed output via a web interface.
It combines:
Backend: Python (Flask)
Data Processing: Pandas-based pipeline
Frontend: HTML, CSS, JavaScript
🏗️ Project Architecture
Frontend (HTML/CSS/JS)

⬇

Flask API (app.py)

⬇

Pipeline Service (pipeline_service.py)

⬇

Processed Output (CSV)
⚙️ Features
📁 Upload CSV file
👀 Preview data (frontend)
🔄 Run data pipeline
🧹 Data validation, cleaning, transformation
📥 Download processed output
⚡ Dynamic and user-driven execution

🔄 Pipeline Flow
Load data
Validate data
Clean data (handle nulls, etc.)
Transform data
Save output file
Pipeline logic is implemented in pipeline_service.py
▶️ How to Run
1. Install dependencies
pip install -r requirements.txt
2. Start the application
python app.py
3. Open in browser
http://127.0.0.1:5000
📤 How to Use
Upload a CSV file
Click Run Pipeline
Download the processed output
🧪 Sample Data Generator
You can generate sample data using:
python generate_data.py
This creates random CSV files for testing
